## ✨ Features

- 🎨 Dark theme with cyan neon accent colors
- ⌨️ Animated typing effect using Typed.js
- 📱 Fully responsive — works on mobile, tablet & desktop
- 🍔 Mobile hamburger navigation menu
- 🎯 Scroll-based active navbar highlighting
- 🖼️ Floating profile image with neon glow animation
- 📊 Animated skill progress bars
- 🏆 Certifications section (HackerRank)
- 💼 Projects showcase with Live Demo & GitHub links
- 📬 Working contact form powered by Formspree
- ⬇️ Resume download button
- 🔗 Social media links (LinkedIn, Instagram, Facebook)
- ✨ Particle background animation on hero section
- 🖱️ Custom cursor glow effect (desktop)

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling, animations & responsive design |
| JavaScript (Vanilla) | Interactivity & DOM manipulation |
| Typed.js | Typing animation effect |
| Boxicons | Icon library |
| Formspree | Contact form backend (no server needed) |
| GitHub Pages | Free hosting & deployment |
